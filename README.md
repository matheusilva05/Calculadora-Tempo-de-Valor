O projeto foi desenvolvido ao pensar em calcular quantos dias necessários eu teria que trabalhar para comprar o produto que eu desejo, pensando que o nosso tempo é valioso. Com isso, foi desenvolvido o
Calculador de Tempo de Vida é uma aplicação web simples que tem como objetivo mostrar quanto tempo de trabalho é necessário para adquirir determinado produto, com base no salário mensal do usuário.

A ferramenta realiza o cálculo convertendo o salário mensal em ganho por dia e por hora, permitindo estimar quantos dias de trabalho são necessários para alcançar o valor desejado. Dessa forma, 
o usuário consegue ter uma percepção mais consciente sobre o custo real de suas compras em relação ao seu tempo de vida.

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, com foco em praticar lógica de programação, manipulação do DOM e criação de interfaces simples e funcionais.
